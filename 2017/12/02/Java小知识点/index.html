Java小知识点

发表于 2017-12-02 | 分类于 Java
  1. ArrayList\<>的菱形括号中必须为完全引用类型。且放入和插入都使用add()，计算长度用size()，set()替换，get()取出。
  2. 插入e到第i位置，可以理解为把第i+1个元素到最后一个元素都往后移一个位置，再把e放到原先第i+1元素的位置。
  3. 比较两个包装器对象时用equals,比较两个枚举类型用= =,
  4. 枚举类型本质上其实就是一个类，枚举中的常量都是该枚举类型的实例。定义枚举类格式，enum XX { }。
  5. 方法重载和重写的区别。重载是一个类中多态性的一种表现，重写是父类与子类之间多态性的一种表现。方法重载是在同一个类中，方法名相同，但具有不同的参数(数量或者类型)和不同的定义，返回类型可以相同也可以不同；；；方法重写相对于子类和父类而言，必须要求重写__从父类继承__的方法方法名，参数表，返回类型都要和父类相同，但方法域里的内容可以修改，注意，static,final,private修饰的方法不能被重写。
  6. 多态的定义：同类的对象对同一消息作出不同的响应就叫做多态。产生条件:(1)有继承关系，(2)子类重写父类方法，(3)父类引用指向子类对象。
  7. 类中的静态成员和方法属于类域，其它属于实例域
  8. 弱类型、强类型、动态类型、静态类型语言的区别：强类型：偏向于不容忍隐式类型转换。譬如说haskell的int就不能变成double；弱类型：偏向于容忍隐式类型转换。譬如说C语言的int可以变成double；静态类型：编译的时候就知道每一个变量的类型，因为类型错误而不能做的事情是语法错误。；动态类型：编译的时候不知道每一个变量的类型，因为类型错误而不能做的事情是运行时错误。譬如说你不能对一个数字a写a10当数组用。
  9. 匿名类形式为：父类或父接口new X = 父类或父接口() {类的主体}。假如只创建这个类的一个对象，就不必为这个类命名，这种类就成为了匿名类。使用匿名内部类的目的：想在这个地点有点什么特殊实现。匿名内部类就是重写父类或接口的方法。匿名类没有名字用的是父类的名字，它只有代码体。
